InesArt

Pintura por números — escolhe um modelo, depois uma cor, e toca no desenho para pintar.

A procurar modelos em Foto/ e Legenda/…
Escolhe uma cor e toca numa área.

Paleta —

Cada número do desenho corresponde à mesma cor aqui. Voltar a clicar numa área já pintada substitui a cor.

Como adicionar um novo modelo

Para o site mostrar um modelo novo, faz commit de dois ficheiros com o mesmo número:

InesarT/
├── Foto/
│   └── 2.png        (ou .jpg / .jpeg — só contornos e números, fundo claro)
└── Legenda/
    └── 2.txt

Conteúdo de Legenda/2.txt:

nome:Cão

1:#39251D
2:#2D2C26
...

A linha nome: é opcional. Os números têm de continuar sequenciais (1, 2, 3…) — se saltares um número, os modelos seguintes deixam de aparecer.

Nota: isto só funciona quando a página é servida por http(s) (GitHub Pages, ou um servidor local) — abrir o ficheiro a fazer duplo-clique não permite carregar as imagens.
